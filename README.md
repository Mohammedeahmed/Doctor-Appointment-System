# Doctor Appointment System UI

## Overview
Designed a user-friendly interface for booking doctor appointments, focusing on clear navigation and smooth user flow.

---

## Problem
Booking doctor appointments can be confusing and time-consuming.

---

## Solution
A simple and intuitive UI that allows users to:
- Select a doctor
- Choose a time slot
- Confirm appointments easily

---

## Features
- Doctor selection
- Time slot selection
- Appointment summary
- Confirmation interaction

---

## Tech Stack
- HTML
- CSS
- JavaScript

---

## UX Focus
- Clear step-by-step flow
- Minimal design for easy navigation
- Reduced user friction in booking process
